<div class="bg-white border-l-4 border-green-600 p-4 rounded-lg mb-6 shadow-sm">
Excel provides regional formatting settings for different styles and languages of presenting information. 
<br><br>
<strong>Regional settings can greatly impact:</strong> <br>
<ul class="list-disc list-outside">
<li>Calendar date formatting</li>
<li>Decimal numbers </li>
<li> Default currency formatting </li>
<li>Formula delimiters
<ul class="list-disc list-outside">
    <li> Symbols used to separate arguments in a function </li>
    <li> Most common symbols for delimiters are commas and semicolons </li>
</ul>
</ul>
</div>

<div class="bg-white border-l-4 border-green-600 p-4 rounded-lg mb-6 shadow-sm">
<h2 class="text-2xl font-semibold text-center mb-4">Changing Format Settings</h2> 
To change the format settings, go to the File menu and click Options. The Options tab will open a dialog box that allows you to choose your regional language settings. 
<br>
<p align="center">
  <img src="{{ '/static/format-settings.png' | relative_url }}" alt="Excel Format Settings example"/>
</p>
<br><br>
<strong>Warning:</strong> Changing the regional format settings will automatically convert any existing values and formulas in your spreadsheet.
</div>
