<h1 class="text-4xl font-semibold text-center mb-4">Formatting Fonts:</h1>

There are various ways to format fonts in Excel. The most common ways are: 
<ul class="list-disc list-inside">
<li>Color </li>
<li>Font name </li>
<li>Size </li>
</ul> <br>

<div class="bg-white border-l-4 border-green-600 p-4 rounded-lg mb-6 shadow-sm">
<h2 class="text-2xl font-semibold text-center mb-4">Font Colors</h2>
The default color for fonts is black. Colors can be applied to fonts by using the “Font color” function. The Font color command remembers the color used last time.

<h2 class="text-2xl font-semibold text-center mb-4">Font Types</h2>
The default font used in Excel is Calibri. Font types can be changed using the font name drop-down menu.

<h2 class="text-2xl font-semibold text-center mb-4">Font Size</h2>
The default font size used in Excel is 11. The size of fonts can be changed using the dropdown menu next to the font name.

<h2 class="text-2xl font-semibold text-center mb-4">Font Characteristics</h2>

Other font characteristics can be adjusted through their commands in the ribbon. These commands are located under the font type and size.  
<br><br>
<p class="font-semibold">Other font characteristics include:</p>
<ol class="list-decimal list-inside">
 <li> Bold
   <ul class="list-disc list-inside">
     <li> applied by either clicking the Bold (B) icon in the Ribbon or using the keyboard shortcut CTRL + B or Command + B </li>
     </ul> </li>
 <li> Italic
   <ul class="list-disc list-inside">
    <li> applied by either clicking the Italic (I) icon or using the keyboard shortcut CTRL + I or Command + I </li> </ul> </li>
 <li> Underlined
 <ul class="list-disc list-inside">
    <li> applied by either clicking the Italic (I) icon or using the keyboard shortcut <strong>CTRL + I or Command + I</strong> </li> </ul> </li>
 <li> Strike through
   <ul class="list-disc list-inside">
    <li> applied by either clicking the Strikethrough (ab) icon or using the keyboard shortcut <strong>CTRL + 5 or Command + Shift + X </strong> </li> </ul> </li>
</ol> <br><br>
