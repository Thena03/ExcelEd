<h2 class="text-2xl font-semibold text-center mb-4"> AVERAGE Function</h2>
The average function is a built-in function that calculates the arithmetic mean (the average). To use this function, use `=AVERAGE`. When calculating the average, you are adding the range and dividing it by the number of values.
<br> <br>

<h2 class="text-2xl font-semibold text-center mb-4"> COUNT Function</h2>
The COUNT function counts cells with numbers in a range. To use this function, use `=COUNT`. This function only counts cells with numbers. <br><br>

  <img src="{{ '/static/COUNT.png' | relative_url }}" alt="Excel Count Fucntion example"/> </p>

<h2 class="text-2xl font-semibold text-center mb-4"> MAX Function</h2> 
The max function finds the highest number in a range. To use this function, use `=MAX`. This function only works for cells with numbers. <br><br>

  <img src="{{ '/static/MAX.png' | relative_url }}" alt="Excel Max example"/> </p>

<h2 class="text-2xl font-semibold text-center mb-4"> MIN Function</h2>
The min function finds the lowest number in a range. To use this function, use `=MIN`. This function only works for cells with numbers. <br><br>
<p align="center">
  <img src="{{ '/static/MIN.png' | relative_url }}" alt="Excel Min example"/> </p>

<h2 class="text-2xl font-semibold text-center mb-4"> MEDIAN Function</h2>
The median function finds the middle value in the data. To use this function, use `=MEDIAN`. <br><br>
