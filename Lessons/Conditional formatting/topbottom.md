<h2 class="text-2xl font-semibold text-center mb-4"> Top Bottom Rules</h2>
The Top/Bottom Rules will highlight cells with one of the following appearance options based on the cell value being the top or bottom values in a range:
<ul class="list-disc list-inside">
<li> Light Red Fill with Dark Red Text </li>
<li> Yellow Fill with Dark Yellow Text </li>
<li> Green Fill with Dark Green Text </li>
<li> Light Red Fill </li>
<li> Red Text </li>
<li> Red Border </li>
</ul>

<strong>Tip:</strong> The default number of items is 10, but you can specify any whole number up to 1000 for Top/Bottom Items to be highlighted.
