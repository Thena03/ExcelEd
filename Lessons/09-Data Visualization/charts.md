<h2 class="text-2xl font-semibold text-center mb-4"> Excel Charts </h2>

Charts are visual representations used to make data more understandable.
<br>
Commonly created charts in Excel are:
<ul class="list-disc list-inside">
<li> Pie chart </li>
<li> Column chart </li>
<li> Line chart </li>
<li> Creating a Chart in Excel </li>
</ul>
<br>
To create a chart in Excel, select a range of cells and then click on the Insert menu and then choose the chart you would like to use to visualize the data.
