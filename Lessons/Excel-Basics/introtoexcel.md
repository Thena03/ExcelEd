## What is Excel?

Excel is a spreadsheet software created by Microsoft used for organizing, analyzing, and visualizing data. This program organizes data in columns and rows.


### Common Functions of Excel:
- Doing math functions
- Data entry
- Data management
- Data Analysis
- Accounting
- Budgeting
- Visualizations

This program runs on every device, including macOS, Windows, Android, and IOS. This allows you to easily use it for work and in your everyday life. The following modules are designed to teach you the basics of Excel and elevate your Excel skills.
