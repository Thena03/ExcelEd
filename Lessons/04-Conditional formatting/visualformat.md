<h1 class="text-4xl font-semibold text-center mb-4"> Visual Conditional Formatting Elements</h1>
<br>
<div class="bg-white border-l-4 border-green-600 p-4 rounded-lg mb-6 shadow-sm"> 
<h2 class="text-2xl font-semibold text-center mb-4"> Data Bars</h2>
Data Bars is a conditional formatting condition that add colored bars to cells in a range to indicate how large the cell values are compared to the other values.

<h2 class="text-2xl font-semibold text-center mb-4"> Color Scales</h2>
Color Scales are a conditional formatting condition used to highlight cells in a range to indicate how large the cell values are compared to other values in the range. 
<br><br>
<p align="center">
  <img src="{{ '/static/color-scales.png' | relative_url }}" alt="Excel Color Scales example"/>

<div class="bg-white border-l-4 border-green-600 p-4 rounded-lg mb-6 shadow-sm"> 
<h2 class="text-2xl font-semibold text-center mb-4"> Icon Sets</h2>
Icon Sets is a conditional formatting condition that adds icons to cells in a range to indicate how large the cell values are compared to the other values in the range. Icon sets can be used together with other conditional formatting rules.
